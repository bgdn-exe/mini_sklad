# Мини-Склад

Учебная конфигурация на платформе **1С:Предприятие 8.3**, созданная в рамках отбора на стажировку 1С-разработчика в КБ99.

## Week 1

Реализовано:

- подсистема `Мини-Склад`;
- справочники `Товары` и `Контрагенты`;
- документы `Поступление` и `Реализация`;
- табличные части с товаром, количеством, ценой и суммой;
- автоматический расчёт суммы строки;
- общий модуль `РаботаСДоками`.

## Week 2

Реализовано:

- справочник «Склады»;
- регистр накопления «ОстаткиТоваров»;
- приходные движения документа «Поступление»;
- расходные движения документа «Реализация»;
- СКД-отчёт «Обороты по товарам» за выбранный период;
- СКД-отчёт «Остатки по товарам» на выбранную дату;
- печатная форма документа «Реализация».

### Дополнительное задание

В демонстрационной конфигурации «Управление торговлей 11.5» выполнена установка и первоначальная настройка модуля интеграции КБ99 ФГИС «Меркурий».

## Как запустить

1. Клонировать репозиторий:

```bash
git clone https://github.com/bgdn-exe/mini_sklad.git
```

2. Создать пустую информационную базу 1С.
3. Открыть её в режиме **Конфигуратор**.
4. Выбрать **Конфигурация → Загрузить конфигурацию из файлов**.
5. Указать папку `src`.
6. Обновить конфигурацию базы данных.
7. Запустить режим **1С:Предприятие**.

## Структура репозитория

```text
mini_sklad/
├── src/
├── .gitignore
└── README.md
```

В папке `src` находится выгрузка конфигурации в XML-файлы.
